# WIN Coaching — Vercel-ready MVP

Deploy this folder as a Vercel project. Add `OPENAI_API_KEY` in Vercel Project Settings → Environment Variables. Optionally add `OPENAI_MODEL`; the default is `gpt-4.1-mini`.

The interface works without the key in Demo Mode. The AI generator activates after the environment variable is added and the project is redeployed.
